Automate Vulnerability Management

Overview

Today’s security and IT operations professionals have too many endpoints to manage and too little time to secure them. As a result, security vulnerabilities can often go days, weeks, and even months before they are repaired. Evolve VM introduces a new, rapid approach to endpoint compliance and vulnerability management that automatically detects and resolves issues at the moment it discovers them. It uses peer-to- peer technology to proactively speed time to resolution of endpoint compliance and vulnerability issues faster than any other product on the market today.

Features

{% include svg/features/health-checks.svg height="20" %}   Endpoint Compliance and Vulnerability Checks

Identify common security holes instantly

Evolve VM is equipped with thousands of pre-packaged endpoint health, compliance, and vulnerability checks that can be run on-demand or on a scheduled basis from the Evolve VM administrator console. These checks cover areas like security configuration checks, software, application, and operating system checks, SCAP-based guidelines, and more.

{% include svg/features/smart-scaling.svg height="20" %}   Real-time Remediation Actions

Drive continuous endpoint compliance and security at scale

Evolve VM delivers real-time remediation actions that can automatically run at scale to fix endpoint health, compliance, and security issues in seconds. From updating configuration settings to turning on application features, Evolve VM offers remediation actions out-of-the-box and the ability to rapidly create new ones as needed without scripting.

{% include svg/features/workflow-designer.svg height="20" %}   Visual Workflow Designer & Engine

Create your own custom checks and remdediations without coding

The Evolve VM Visual Workflow Designer and Engine grants teams the ability to create custom compliance and security checks and remediation actions in seconds without writing a single a line of code. Evolve VM administrators can use the Workflow Designer and Adaptiva’s built-in API creation capabilities to easily integrate with other third-party products for deeper customization.

{% include svg/features/visual-reporting.svg height="20" %}   Interactive Dashboard System

Gain instant insights on endpoint health and security success

Evolve VM Interactive Dashboard System provides a live view of vulnerabilities and the risk level across an organization. Choose from dozens of interactive dashboards to get real-time information on which checks and remediation actions were run, their current status, success, failure, and repair rates, and trends over time.
